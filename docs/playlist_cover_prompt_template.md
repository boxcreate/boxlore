# Daily News Playlist Cover Prompt Template

This is the default template used to generate the cover artwork for the daily news playlist (**"the boxcast brief"**). It is designed to visually synthesize the day's top 3 headlines without generating any text, labels, or numbers.

## Default Template

```text
A professional, minimalist, high-contrast podcast cover art for a daily news playlist. The design should visually synthesize the following key news stories of the day into a cohesive, symbolic illustration:
1. {HEADLINE_1}
2. {HEADLINE_2}
3. {HEADLINE_3}

The style must be clean, abstract, and editorial (similar to premium news publication covers). Use a modern, limited color palette with a dark background. Combine the core visual concepts of these stories into a single fluid, creative composition without structural layout boundaries. Do not include any text, letters, or labels on the image.
```

## Recommended Model
* **Hugging Face Model:** `black-forest-labs/FLUX.1-schnell` (provides the best visual synthesis and layout composition).
